# Привет, меня зовут Мария 👋


### Обо мне:
Я начинающий BI/Data аналитик.

Работаю в IT с 2011 года. Математическое образование, 4+ года в Project Management (сбор требований, ТЗ, интеграции) помогают мне быстро погружаться в бизнес-задачи и работать с данными осознанно. 

В 2023 году целенаправленно перешла в аналитику данных — нравится искать закономерности и превращать цифры в понятные решения для бизнеса.

### 🛠️ Инструменты:
- **Анализ данных:** Python (Pandas, NumPy, Matplotlib, Seaborn, Jupyter), SQL (PostgreSQL, ClickHouse)
- **Визуализация:** Datalens, Superset, Insight BI
- **Инфраструктура:** Git, Docker (учебный проект: развернула локальный стенд с БД и BI)
- **Продуктовая аналитика:** опыт в рамках курса-симулятора (метрики DAU, MAU, WAU, CTR, Retention, A/A и A/B-тесты, построение ETL-пайплайна, настройка алертов)
