# Привет, меня зовут Мария 👋

Я аналитик данных / BI-аналитик с математическим образованием и опытом в Project Management.

В IT с 2011 года. Больше 4 лет управляла проектами в веб-разработке: собирала требования, писала ТЗ, разбиралась в интеграциях. 
После перешла в аналитику. Теперь мой фокус — помогать в принятии обоснованных решений на основе данных.

Мне важно видеть результат своей работы. Когда полученный результат реально помогает — это лучшая мотивация.

Что умею:
- Собирать и формализовывать требования (привет, прошлый опыт!)
- SQL: вложенные запросы, оконные функции, CTE, конструкции case
- Python: очистка и предобработка данных, EDA
- Рассчитывать метрики и проверять гипотезы
- Понятно визуализировать результаты и делать выводы
- Работа над проектами в системе контроля версий git

В свободное время рисую акварелью 🎨
